# WHAT IS THIS

* This is a repo of a simple API request using rest-client gem for Ruby


# Dependencies
  
* Ruby-version 2.5.1
* Rest-client 2.0.2
* Pry
* Other dependencies of the rest-client


# Usage


    cd /path/to/rest_search
    
    gem install bundler
    
    bundle install

    ruby rest_search.rb
